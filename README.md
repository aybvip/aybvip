## What I'm Building

### Pi Visual Builder
An AI-Driven No-Code Platform for Pi App Studio — combining the power of an AI Agent with professional visual development tools.

- Visual drag-and-drop canvas with Elements Tree
- Step-by-step workflow engine (When → Do)
- Built-in relational database with privacy rules
- AI Agent that builds apps from natural language chat
- Plugin marketplace for community components

**Repository**: [Pi-Visual-Builder](https://github.com/aybvip/Pi-Visual-Builder)

### PiDCTP (PiRC3)
Decentralized Commerce Trust Protocol — on-chain trust infrastructure for Pi Network:

- Escrow contracts (milestone & group)
- Reputation system with Soulbound Badges
- Dispute resolution with decentralized arbitration
- Built on Rust/Soroban for Stellar

**Pull Request**: [PiNetwork/PiRC #378](https://github.com/PiNetwork/PiRC/pull/378)
